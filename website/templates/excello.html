Running Excello

To use Excello, the code must be running somewhere, and Excel must be given a manifest file that points to where this is running so it can provide it as an add-in. You can run Excello on your own computer, or use an online hosted version. The online version should be easier but runs a little slower.

Using an externally hosted version

This was written for using Excel online, steps may differ slightly for different version of Excel.
  • Download this manifest.xml file
  • Open this spreadsheet (in an Excel version that supports add-ins)
  • Insert > Office Add-ins > Upload My Add-in
  • Find and upload the manifest.xml file you downloaded
  • Excello will now be available in your home tab in the ribbon

Running Locally

  • Clone the Github Repo
  • Run with Node (helpful Microsoft tutorial)
  • Open Excel (a version that supports add-ins)
  • Insert > Office Add-ins > Upload My Add-in
  • Find and upload the local_manifest.xml file you downloaded
  • Excello will now be available in your home tab in the ribbon
